# Small Agentic AI Learning Projects

This repository is a home for small, focused projects for learning agentic AI concepts.

The goal is to keep each project simple, isolated, and easy to experiment with. That makes it useful for trying out ideas like tool calling, planning, memory, orchestration, and agent workflows without turning the repo into a large application.

## What goes here

- Small proof-of-concept agent projects
- Tool-calling experiments
- Prompt and workflow prototypes
- Notes or scripts that support learning agent behavior

## Current Structure

- `tool_calling/` - early experiment area for tool-calling work

## How to Use

Each subproject can define its own setup and run instructions. For now, this repo is a lightweight workspace for building and testing ideas as they come up.

## Conventions

- Keep projects small and focused
- Prefer clear, minimal examples over production-style complexity
- Add a short README inside a subproject if it needs specific instructions
